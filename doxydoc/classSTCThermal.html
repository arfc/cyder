Detailed Description

STCThermal is a skeleton component model that does nothing.

This disposal system component will do nothing. This Component is intended as a skeleton to guide the implementation of new Components.

The STCThermal model can be used to represent components of the disposal system such as the Waste Form, Waste Package, Buffer, Near Field, Far Field, and Envrionment.
