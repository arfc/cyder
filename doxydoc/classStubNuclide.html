Detailed Description

StubNuclide is a skeleton nuclide model model that does nothing.

This disposal system nuclide model will do nothing. This NuclideModel is intended as a skeleton to guide the implementation of new NuclideModels.

The StubNuclide model can be used to represent nuclide models of the disposal system such as the Waste Form, Waste Package, Buffer, Near Field, Far Field, and Envrionment.
